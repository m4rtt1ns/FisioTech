Guia rápido para contribuições

1. Abra uma issue descrevendo a mudança / bug.
2. Faça um branch com nome significativo: feature/descrição ou fix/descrição.
3. Adicione testes para novas regras de negócio.
4. Mantenha o estilo PEP8 e rode `flake8` / `black` se preferir.
5. Abra Pull Request com descrição clara e referência à issue.
